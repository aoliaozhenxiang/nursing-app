# 养老护理员职业技能等级认定考试培训助手 - 设计规范

## 产品定位

**产品名称**: 养老护理员考证助手
**考试类型**: 养老护理员职业技能等级认定考试 四级（中级工）
**核心理念**: 教、学、练、考一体化，提供沉浸式学习体验

## 设计风格

### 视觉风格: 柔和卡片风 (Soft Card UI)
- **感觉**: 温暖、专业、友好、易于学习
- **适用场景**: 在线教育、技能培训、考试备考

### 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #6C63FF | 薰衣草紫，代表专业与信任 |

| 文字次 | #636E72 | 次要文字 |

## 功能架构

### 整体架构: 五大模块

```
┌─────────────────────────────────────────────────┐
│                    首页                          │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐            │
│  │ 理论学  │ │ 实操练  │ │ 考题考  │            │
│  │ (视频)  │ │ (图片)  │ │ (刷题)  │            │
│  └─────────┘ └─────────┘ └─────────┘            │
│  ┌─────────────────────────────────────────┐   │
│  │           学习进度总览                    │   │
│  └─────────────────────────────────────────┘   │
│  ┌─────────────────────────────────────────┐   │
│  │           继续学习                      │   │
│  └─────────────────────────────────────────┘   │
├─────────────────────────────────────────────────┤
│ 理论学习 │ 实操练习 │ 刷题 │ 我的                │
└─────────────────────────────────────────────────┘
```

### 模块1: 理论学习 (Theory)
- **章节列表**: 按教材章节组织
- **视频教学**: 上传/播放理论教学视频
- **PPT课件**: 支持PPT文件预览
- **记忆游戏**: 闯关式记忆训练
- **先学后练**: 学习完成才能进入练习

### 模块2: 实操练习 (Practice)
- **操作图片库**: 标准操作图片展示
- **图片对比评判**: 用户上传练习图片与标准对比
- **视频演示**: 标准操作视频
- **步骤分解**: 逐步学习操作要点

### 模块3: 刷题考试 (Exam)
- **题库练习**: 单选、多选、判断、简答
- **章节练习**: 按章节刷题
- **模拟考试**: 全真模拟考试环境
- **错题本**: 自动收集错题
- **收藏夹**: 收藏重点题目

### 模块4: 闯关记忆 (Memory)
- **记忆卡片**: 翻转式记忆
- **闯关模式**: 逐关解锁
- **限时挑战**: 时间压力训练
- **知识点关联**: 关联相关题目

### 模块5: 我的 (Profile)
- **学习进度**: 各模块完成情况
- **收藏夹**: 收藏内容
- **错题本**: 错题回顾
- **设置**: 主题、通知等

## 页面清单

### 首页 (index)
- 顶部: 欢迎语 + 学习天数
- 中部: 三大功能入口卡片
- 下部: 学习进度总览 + 继续学习

### 章节列表页 (chapters)
- 章节选择列表
- 章节完成状态
- 进入各章节详情

### 章节详情页 (chapter-detail)
- Tab切换: 理论学习 | 实操练习 | 刷题
- 章节内容展示
- 学习进度

### 理论学习页 (theory)
- 视频播放区
- PPT预览区
- 学习笔记
- 完成标记

### 实操练习页 (practice)
- 标准图片展示
- 用户上传对比区
- 评判结果展示
- 操作要点

### 刷题页 (exam)
- 题目展示区
- 选项区
- 答案解析区
- 进度条

### 闯关记忆页 (memory)
- 关卡列表
- 记忆卡片
- 挑战计时
- 成就展示

## 交互设计

### 卡片设计
- 新拟态双层阴影
- 大圆角 (borderRadius: 24)
- 图标容器 48x48

### 按钮设计
- 渐变按钮: LinearGradient (#6C63FF → #896BFF)
- 胶囊形状 (borderRadius: 9999)

### 进度展示
- 进度环 (ProgressRing)
- 进度条 (ProgressBar)
- 完成徽章

### 动画效果
- 按压反馈: scale 1 → 0.97
- 完成激活: scale 0 → 1.15 → 1.0

## 技术实现

### 前端
- Expo 54 + React Native
- Expo Router 路由
- TailwindCSS + Uniwind 样式
- expo-av 音视频
- expo-image-picker 图片上传
- react-native-svg 进度环

### 后端
- Express.js
- RESTful API
- 数据存储

- ❌ 禁止白色卡片背景（应用 #F0F0F3）
